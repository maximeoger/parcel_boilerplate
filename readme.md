# Parcel Boilerplate

Simple boilerplate for small project or SPA, built with [parcel](https://parceljs.org/)

## installation

1. `git clone https://github.com/maximeoger/parcel_boilerplate.git`
2. `npm install`
3. `npm run start`

## Modules

- babel
- autoprefixer
- node-sass

## build

- npm run build
